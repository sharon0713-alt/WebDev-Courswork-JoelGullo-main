# Unit 1 Reflection

I learned about the history of the web, how websites work, and the basics of HTML in Unit 1.

I learned that the Internet began in the 1960s and 1970s as ARPANET, a network that the US Department of Defense made to survive an attack. It sent data using packet switching. Tim Berners-Lee made the Internet available to everyone by creating the World Wide Web in the 1990s. He made the first web server and web browser. Later, web browsers like Netscape and Internet Explorer, search engines like Yahoo, and sites like Facebook and Amazon made the web even more popular.

Then I learned about developers, websites, and web browsers. Users can get to websites through browsers, and developers write code to make them. I also looked "behind the curtain" to see how HTML makes web pages.

After that, I learned the basics of how HTML works. The first thing on every page is <!DOCTYPE html>, then <html>, <head>, <title>, and <body>. To make headings, use <h1> to <h6> and to make paragraphs, use <p>. I did a code challenge where I changed the content of a boring website to make it more interesting.

I also learned how to use text formatting tags. For example, <strong> and <b> make text bold, <em> and <i> make it italic, <u> underlines, <del> strikes through, <ins> inserts, <small> makes text smaller, <mark> highlights, <br> adds a line break, and <hr> adds a horizontal line. These tags are important for screen readers and help organize content.

Lastly, I learned how to use <!-- comment --> to add comments in HTML. Comments let me write down what my code does without changing the web page.

In general, Unit 1 taught me a lot about the history of the web and the basics of HTML. I can now make simple web pages that have headings, paragraphs, and text that is formatted.
